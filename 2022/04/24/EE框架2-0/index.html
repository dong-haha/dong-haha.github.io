EE框架2.0

发表于 2022-04-24 分类于 前端 ， 桌面应用开发

一个入门简单、跨平台桌面软件开发框架
github：https://github.com/wallace5303/electron-egg
原文档： https://www.yuque.com/u34495/mivcfg

特性

  1. 跨平台：一套代码，可以打包成windows版、Mac版、Linux版运行
  2. 简单高效：只需学习js语言，同时支持vue、react、html等前端技术
  3. 前端独立：理论上支持任何前端技术，编写出精美的UI效果
  4. 工程化：可以用服务端的开发思维，来编写桌面软件
  5. 高性能：事件驱动、非阻塞式IO
  6. 功能丰富：服务端的技术场景等
  7. 功能demo：桌面软件常见功能，后续逐步集成或提供demo
  8. 更多功能请看文档

该框架是在electron基础上进行修改，属于半成品的桌面应用，提供很多demo，方便快速成型一个桌面应用，效率很高 。

目录

注意：

  1. front: 前端项目的目录，进去后就是vue 的node_modules
  2. main.js： electron项目入口，原生的electron代码这个项目里没有，需要 const Appliaction = require(‘ee-core’).Appliaction; 作者把electron代码放在ee-core包里，这里几乎都是项目的配置文件。
  3. public: vue项目带包后移到这里
  4. build 可以添加其他资源，将一起附带到安装包
